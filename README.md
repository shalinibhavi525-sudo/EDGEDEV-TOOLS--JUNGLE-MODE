## I. THE GENESIS: ARCHITECTING FROM THE EDGE

This protocol was not designed in a laboratory. It was forged in the **Teliamura Reserved Forest**, at a **Border Security Force (BSF)** sector headquarters in Tripura, India. 

As a 17-year-old developer living within a military perimeter, I operate under **Adversarial Network Conditions**:
- **Primary Uplink:** High-latency, non-persistent 4G via a mobile hotspot (The only available tether).
- **Power Infrastructure:** Daily grid cycles necessitating development on internal battery reserves.
- **Data Scarcity:** A 2GB daily threshold where a single background update can terminate a week's worth of progress.

**EdgeDev** is a bespoke hardening protocol for VS Code. It silences the "telemetric noise" of modern IDEs to preserve the integrity of the developer's most limited resource: **Bandwidth.**
